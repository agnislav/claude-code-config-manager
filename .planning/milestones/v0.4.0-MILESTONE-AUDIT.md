## What Was Built

### Phase 6: Lock UX Rework (2026-02-19)
User scope locks by default on activation — write operations blocked before user interaction. Lock icons use state semantics (`$(lock)` shown when locked). Toggle moved from inline scope node to toolbar at position left of filter.

### Phase 7: Collapse/Expand Toolbar Buttons (2026-02-20)
Collapse All delegates to VS Code built-in. Expand All recursively walks tree and reveals each expandable node. Both toolbar-only (hidden from command palette). Toolbar order: lock, collapse, expand, filter.

### Phase 8: Object Settings Expansion (2026-02-20)
Object-type settings render as expandable tree nodes with collapse/expand arrows. Child nodes display key as label and formatted value as description. Scalar/array settings unchanged. Single-level expansion only.
