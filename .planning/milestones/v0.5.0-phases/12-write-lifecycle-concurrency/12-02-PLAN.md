**Deactivation cleanup**

Change `deactivate()` from synchronous to async:
```typescript
export async function deactivate(): Promise<void> {
  // Clear all tracked editor-tree sync timeouts
  for (const [key, timeout] of syncTimeouts) {
    clearTimeout(timeout);
    syncTimeouts.delete(key);
  }

  // Wait for any in-flight writes to complete (poll every 50ms, max 5 seconds)
  const maxWait = 5000;
  const pollInterval = 50;
  let waited = 0;
  while (getInFlightWriteCount() > 0 && waited < maxWait) {
    await new Promise(resolve => setTimeout(resolve, pollInterval));
    waited += pollInterval;
  }

  // Disposables handled by context.subscriptions
}
```

Note: `syncTimeouts` must be declared at module scope (outside `activate`) so `deactivate` can access it. Move it to module scope alongside where `syncTimeout` is currently declared, or restructure so the map is accessible from both functions.
