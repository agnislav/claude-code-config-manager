**Step 3: Create a helper function in builder.ts**

Add a helper function `buildOverlapAccessibilityLabel` near the other helper functions (after `applyOverrideSuffix`). This function appends overlap status text to an accessibility label:

```typescript
function buildOverlapAccessibilityLabel(base: string, overlap: OverlapInfo): string {
  const parts = [base];
  if (overlap.overrides) {
    parts.push(`overrides ${SCOPE_LABELS[overlap.overrides.scope]} scope value`);
  }
  if (overlap.isOverriddenBy) {
    parts.push(`overridden by ${SCOPE_LABELS[overlap.isOverriddenBy.scope]} scope`);
  }
  if (overlap.duplicates) {
    parts.push(`duplicates ${SCOPE_LABELS[overlap.duplicates.scope]} scope value`);
  }
  if (overlap.isDuplicatedBy) {
    parts.push(`duplicated in ${SCOPE_LABELS[overlap.isDuplicatedBy.scope]} scope`);
  }
  return parts.join(', ');
}
```

**Step 4: Add accessibilityInformation to every leaf node builder method**

For each leaf node build method in builder.ts, add `accessibilityInformation` to the returned VM object. Use `SCOPE_LABELS[scopedConfig.scope]` for scope name. The label format for each node type:

**PermissionRuleNode** (in `buildPermissionRule`):
- Label: `"Permission rule: {category} {rule}, {scopeLabel} scope"` + overlap suffix
- Example: `"Permission rule: allow Bash(curl *), User scope"`
- Add to the return object: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Permission rule: ${category} ${rule}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**EnvVarNode** (in the `buildEnvVars` map callback):
- Label: `"Environment variable: {key} equals {value}, {scopeLabel} scope"` + overlap suffix
- Truncate value to 50 chars for readability
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Environment variable: ${key} equals ${value.length > 50 ? value.substring(0, 50) + '...' : value}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**McpServerNode** (in `buildMcpServers` map callback):
- Label: `"MCP server: {name}, {type} transport, {scopeLabel} scope"` where type is "sse" or "stdio"
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`MCP server: ${name}, ${isSseConfig(config) ? 'sse' : 'stdio'} transport, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**PluginNode** (in `buildPlugins` map callback):
- Label: `"Plugin: {pluginId}, {enabled ? 'enabled' : 'disabled'}, {scopeLabel} scope"` + overlap suffix
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Plugin: ${pluginId}, ${enabled ? 'enabled' : 'disabled'}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**SettingNode** (in `buildSettingVM`):
- Label: `"Setting: {key} equals {formatValue(value)}, {scopeLabel} scope"` + overlap suffix
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Setting: ${key} equals ${formatValue(value)}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**SettingKeyValueNode** (in `buildSettingKeyValueVM`):
- Label: `"Setting property: {parentKey}.{childKey} equals {formatValue(value)}, {scopeLabel} scope"` + overlap suffix
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Setting property: ${parentKey}.${childKey} equals ${formatValue(value)}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**SandboxPropertyNode** (in `buildSandboxPropertyVM`):
- Label: `"Sandbox property: {key} equals {formatValue(value, 'raw')}, {scopeLabel} scope"` + overlap suffix
- For expandable objects use `"Sandbox property: {key}, {scopeLabel} scope"`
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Sandbox property: ${key}${isExpandableObject ? '' : ' equals ' + formatValue(value, 'raw')}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**SandboxPropertyNode child** (in `buildSandboxChildVM`):
- Label: `"Sandbox property: {parentKey}.{childKey} equals {formatValue(value, 'raw')}, {scopeLabel} scope"` + overlap suffix
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Sandbox property: ${parentKey}.${childKey} equals ${formatValue(value, 'raw')}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`

**HookEntryNode** (in `buildHookEntryVM`):
- Label: `"Hook: {hook.type} for {eventType}{matcherPattern ? ' matching ' + matcherPattern : ''}, {scopeLabel} scope"` + overlap suffix
- Add: `accessibilityInformation: { label: buildOverlapAccessibilityLabel(\`Hook: ${hook.type} for ${eventType}${matcherPattern ? ' matching ' + matcherPattern : ''}, ${SCOPE_LABELS[scopedConfig.scope]} scope\`, overlap) }`
