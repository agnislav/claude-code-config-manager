**In `src/commands/addCommands.ts`**, add a new command registration inside `registerAddCommands()` for `claudeConfig.addSetting`. Follow the exact same pattern as `addPermissionRule`, `addEnvVar`, `addHook` commands. The flow:

1. Resolve file path using `resolveFilePath(node, configStore)`.
2. Get the config for this file path to determine already-set keys. Use `configStore.getAllScopes()` to find the ScopedConfig matching the resolved filePath, then read its `config` object. Filter out keys in `DEDICATED_SECTION_KEYS` to get `existingSettingKeys`.
3. Build QuickPick items from `KNOWN_SETTING_KEYS`, filtering out keys already present in `existingSettingKeys`. Each item: `{ label: key, description: SETTING_TYPE_MAP[key] ?? 'unknown' }`.
4. Add a special last item: `{ label: '$(edit) Enter custom key...', value: '__custom__', alwaysShow: true }` with a separator (`kind: vscode.QuickPickItemKind.Separator`) before it.
5. Show QuickPick with `placeHolder: 'Select setting to add'`.
6. If user picks the custom key option, show an InputBox: `{ prompt: 'Enter setting key', validateInput: v => v.trim() ? null : 'Key cannot be empty' }`. If cancelled, return.
7. Determine the value type: look up `SETTING_TYPE_MAP[selectedKey]`. If not found (custom key), default to `'string'`.
8. Prompt for value based on type:
   - **boolean**: Show QuickPick with `[{ label: 'true', value: true }, { label: 'false', value: false }]`, placeHolder `'Set value for {key}'`.
   - **string**: Show InputBox with `prompt: 'Enter value for {key}'`. Allow empty string (don't validate non-empty).
   - **number**: Show InputBox with `prompt: 'Enter value for {key}'`, `validateInput` checking `isNaN(Number(v))` returns `'Must be a number'`. Parse with `Number()`.
   - **string[]**: Show InputBox with `prompt: 'Enter comma-separated values for {key}'`. Split by comma, trim each, filter empty strings. Result is an array.
   - **object**: Show InputBox with `prompt: 'Enter JSON value for {key}'`, `validateInput` that tries `JSON.parse(v)` and returns error message on failure. Parse with `JSON.parse()`.
9. If user cancels value input, return.
10. Write using `withWriteRetry(filePath, () => setScalarSetting(filePath, selectedKey, value))`.

Import `KNOWN_SETTING_KEYS`, `SETTING_TYPE_MAP`, `DEDICATED_SECTION_KEYS` from constants. Import `setScalarSetting` from configWriter.

Note: For the QuickPick items, define a local interface extending `vscode.QuickPickItem` with an optional `value` field to carry the key string, similar to how other commands use `{ label, value }` pattern. The separator item uses `kind: vscode.QuickPickItemKind.Separator`.
