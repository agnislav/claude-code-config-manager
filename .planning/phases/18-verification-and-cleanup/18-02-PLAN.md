   - User with `{ env: { API_KEY: 'user-key' } }`, ProjectShared with `{ env: { API_KEY: 'project-key' } }`
   - Find User scope's EnvVar 'API_KEY'
   - Assert: overridden by ProjectShared

   - Assert: `nodeContext.scope === ConfigScope.ProjectShared`
